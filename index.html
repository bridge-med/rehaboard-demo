データ管理

エクスポート: 現在の全データ(施設・PT・目標・実績・担当患者・月次レビュー)を JSON で書き出し、別端末への移行や定期バックアップに使えます。
インポート: 書き出した JSON を読み込んで状態を復元します。
リセット: 全データをクリアして初期状態に戻します(取り消し不可)。

ライセンス情報

フル機能版で稼働中です。データはこのブラウザの LocalStorage に保存されています。
機種変更時は「JSON エクスポート」でバックアップ→新ブラウザで「JSON インポート」してください。

Demo Mode — データはリロードで消えます。フル機能版を購入する(¥2,980)

RehaBoard へようこそ — 初期設定

ステップ 1 / 4

1. 施設情報

施設名を入力してください。後から「設定」画面で変更できます。

2. PT 名簿

担当 PT の名前を入力してください。実名・略称どちらでも構いません(院内管理なので、好きな表記で)。

3. 当月の目標

当月の部門目標を設定してください。月の途中で変更したい時は「目標設定」画面でいつでも変えられます。

4. サンプルデータ

動作確認用のサンプルデータが入っています。実運用で使う場合は削除してから始めるのがおすすめです。
